# Chrome 恐龙跳跃游戏 PRD

## 项目概述
复刻 Chrome 浏览器断网时的 T-Rex 恐龙跳跃游戏，支持移动端优先体验。

## 游戏核心机制

### 1. 基础玩法
- **角色**：像素风格 T-Rex 恐龙（类似 Chrome 原版）
- **操作**：
  - 空格键 / 点击屏幕 / 触摸：跳跃
  - 下箭头键 / 下滑手势：下蹲（躲避翼龙）
- **目标**：躲避障碍物，获得尽可能高的分数

### 2. 障碍物系统
- **仙人掌**：不同高度（小、中、大），随机出现
- **翼龙**：飞行障碍物，需要下蹲躲避
- **出现规律**：随速度增加，障碍物密度增加

### 3. 游戏机制
- **速度递增**：游戏速度随时间/分数逐渐增加
- **昼夜交替**：每 700 分切换白天/黑夜模式
- **分数系统**：
  - 每 100 分有音效提示
  - 显示当前分数和最高分（localStorage 存储）
- **碰撞检测**：像素级碰撞检测

### 4. 视觉效果
- **像素风格**：8-bit 复古风格
- **动画**：
  - 恐龙奔跑动画（腿部摆动）
  - 跳跃抛物线
  - 障碍物移动
  - 云朵背景（视差滚动）
- **昼夜模式**：
  - 白天：白色背景，黑色恐龙
  - 黑夜：黑色背景，白色恐龙

## 移动端优化

### 1. 触控操作
- **点击屏幕任意位置**：跳跃
- **向下滑动手势**：下蹲
- **长按**：持续下蹲（用于躲避连续翼龙）

### 2. 响应式布局
- 游戏画布占满屏幕宽度
- 高度自适应，保持合适比例
- 一屏显示，无需滚动

### 3. 视觉优化
- 大按钮/大触控区域
- 清晰的分数显示
- 游戏结束界面简洁明了

## 技术实现

### 文件结构
```
/home/yuan/dev/game-hub/dino.html
```

### 核心功能模块
1. **Game 类**：主游戏逻辑
2. **Dino 类**：恐龙角色（跳跃、下蹲、动画）
3. **Obstacle 类**：障碍物（仙人掌、翼龙）
4. **Cloud 类**：背景云朵
5. **Collision 检测**：碰撞检测算法
6. **Score 管理**：分数计算和存储

### 游戏循环
- requestAnimationFrame 实现流畅动画
- 60 FPS 目标帧率
- 时间差分计算保证速度一致性

## 游戏状态
- **等待开始**：显示 "点击开始" 提示
- **游戏中**：正常运行
- **游戏结束**：显示分数，可重新开始
